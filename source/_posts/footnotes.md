{% dropcap "脚" %}注是学术写作与旧式报章中不可或缺的元素。本刊的脚注系统由两个标签配合完成：正文中的 {% marginalia "引用标记" "使用 {% fn %} 标签生成上标数字，点击跳转至文末注释。" %} 与文末的注释列表。读者可点击上标数字跳至注释，亦可点击注释末尾的回引符号返回正文。

## 基本用法

在正文中需要加注的词语后方插入 `{% fn "1" "1" %} `标签，然后在文末使用 `{% fnitem %}` 写出具体注释内容。维多利亚时代的学者们常在书籍页边或章末添加大量脚注{% fn "2" "2" %}，有时注释的篇幅甚至超过正文本身。

{% paranote "citation" "此段参考了1892年版《大英百科全书》的脚注规范。" "left" %}
{% endparanote %}

## 编号方式

本刊支持两种编号方式：

**语义化编号**（推荐）——使用有意义的标签作为锚点 ID，方便日后查阅和维护：

- 第一个参数为锚点 ID：`{% fn "mcluhan" %}`
- 第二个参数为显示的数字（可选）：`{% fn "mcluhan" "①" %}`

若省略第二个参数，则直接显示 ID。

## 完整示例

麦克卢汉在其著作中提出了 {% fn "mcluhan" "1" %} 媒介即讯息的理论，认为媒介本身的形式比其传递的内容更为重要。这一观点在广播时代得到了充分验证{% fn "broadcast" "2" %}。后来的学者进一步指出，数字时代的算法推荐彻底消灭了偶然性{% fn "algorithm" "3" %}——而这恰恰是传统媒介最珍贵的特质之一。

{% endparanote %}

## 不带编号的脚注

若只需在底部添加说明而不需要数字标记，可省略显示参数：

维多利亚时期的印刷商{% fn "printer" %}通常会在书籍扉页印上精美的出版社徽章。
