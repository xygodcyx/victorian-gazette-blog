{% dropcap "旁" %}批，又称边际注，是维多利亚时代书籍中最常见的辅助阅读手段。读者或编辑在书页空白处写下对某个词语的解释、评论或联想，后来的读者便可从中获益。本刊以数字手段还原了这一传统{% fn "digital-margin" "1" %}——被标记的词语以虚线标示，鼠标悬停时注释框浮现于侧。

{% endparanote %}

<!-- more -->

## 基本用法

在正文中，使用 {% marginalia "被批注的词语" "此处写注释内容" %} 即可创建一个旁批标记。被标记的词语将显示为暗红色斜体并带有金色虚线下划线，提示读者此处有额外信息。将鼠标悬停于词语之上，右侧便会浮现一个羊皮纸色的注释框，以虚线连接至原文。

{% paranote "citation" "参考：19世纪英国诗人柯勒律治以在书籍页边写满批注而闻名，其旁批后来被结集出版。" "left" %}
{% endparanote %}

## 实际应用

本刊所使用的正文字体为 {% marginalia "Cormorant Garamond" "一款免费的旧式衬线字体，由 Christian Thalmann 设计，保留了传统印刷品的温润质感与优雅比例。" %}，标题字体为 {% marginalia "Abril Fatface" "一款灵感源自19世纪海报的粗衬线字体，适用于醒目的大标题。" %} 与 {% marginalia "UnifrakturMaguntia" "一款哥特体字型，源自中世纪手稿传统，在本刊中用于报头与装饰性小标题。" %}。这三种字体的组合构成了本刊独特的视觉风格。

在维多利亚时代的印刷史中，{% marginalia "卡斯隆活字" "William Caslon 于18世纪设计的经典活字，在整个维多利亚时期被广泛使用于书籍印刷。" %} 与 {% marginalia "博多尼活字" "Giambattista Bodoni 设计的现代衬线体，以其强烈的粗细对比著称。" %} 是最具代表性的两款字体。它们一个温润含蓄，一个锐利张扬，恰如英国与欧陆两种审美传统的对话。

## 旁批与正文的关系

好的旁批应当：

- 补充信息而不打断阅读节奏
- 篇幅适中，一两句话即可
- 与正文形成对话，而非简单重复

本刊的旁批系统在移动设备上会自动转为内联展开模式——注释不再浮于侧边，而是直接插入正文段落之间，以金色左边框标识。无论在桌面还是手机上，旁批都能优雅地呈现。
