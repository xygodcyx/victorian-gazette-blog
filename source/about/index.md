## 关于本站

### 建站初衷

这个博客创建于 2024 年，最初是为了记录技术学习过程中的心得和笔记。随着内容的不断积累，它逐渐成为了我与读者交流、分享知识的重要平台。

### 技术栈

本站使用以下技术构建：

| 组件     | 技术选型                |
| -------- | ----------------------- |
| **框架** | [Hexo](https://hexo.io) |
| **主题** | 自定义维多利亚风格主题  |
| **托管** | GitHub Pages / VPS      |
| **评论** | Giscus / Waline         |
